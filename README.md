## 99. 참고
### 1. build 결과 배포
1. 패키지 인스톨
   ```
   npm i
   ```
2. 프로젝트 빌드
   ```
   npm run build
   ```
3. 빌드 결과물 기준 서버 실행
   ```
   npx serve -sn build
   ```
